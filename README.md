# 📊 Data Cleaning Project (Excel)

## 📌 Overview
This project demonstrates the process of cleaning and preparing a raw dataset using Microsoft Excel. The dataset initially contained missing values, duplicate records, and inconsistent formatting, which were handled to improve data quality and usability.

---

## 📁 Dataset
- File: `Cleaned_data.xlsx`
- Type: Structured dataset after preprocessing
- Tool used: Microsoft Excel

---

## 🔧 Data Cleaning Steps Performed

### 1. Handling Missing Values
- Identified blank/null cells
- Replaced or removed missing data where necessary

### 2. Removing Duplicates
- Used Excel's built-in feature to eliminate duplicate rows
- Ensured each record is unique

### 3. Data Formatting
- Standardized date formats
- Cleaned text fields (removed extra spaces, fixed casing)
- Ensured consistent data types across columns

### 4. Data Correction
- Fixed inconsistent or incorrect entries
- Validated values to maintain accuracy

### 5. Column Optimization
- Renamed column headers for better understanding
- Removed unnecessary or redundant columns

---

## 🎯 Objective
To convert raw, unstructured data into a clean and structured format suitable for:
- Data Analysis  
- Reporting  
- Visualization
